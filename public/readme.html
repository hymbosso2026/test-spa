Project Overview

Welcome to NKY Travel, a modern and interactive web application designed to inspire travelers and showcase beautiful destinations around the world. This project combines elegant design with powerful features to create an immersive travel storytelling experience.

"A visual journey that connects travelers, shares moments, and celebrates the beauty of exploration."

Key Features

🎬 Camera Capture

Capture beautiful moments directly from your device's camera and share them instantly.

🗺️ Travel Mapping

Explore destinations and view travel routes with an interactive mapping interface.

📸 Gallery Management

Organize and showcase your travel photos in beautiful, responsive galleries.

✍️ Travel Notes

Document your journeys with detailed travel notes and observations.

🌟 Spotboard

Discover and share hidden gems and travel recommendations.

👥 Community

Connect with other travelers and explore their adventures.

Technology Stack

React 18 TypeScript Vite Tailwind CSS Supabase PostgreSQL
  • Frontend: React with TypeScript for type-safe, scalable components
  • Build Tool: Vite for lightning-fast development and optimized production builds
  • Styling: Tailwind CSS for utility-first, responsive design
  • Backend: Supabase with PostgreSQL for real-time data management
  • APIs: RESTful architecture with secure authentication

Project Structure

Main Directories:

  • src/modules/ - Core feature modules
    • overview/ - Dashboard and overview components
    • presentation/ - Authentication and welcome screens
    • spotboard/ - Photo spotting and sharing
    • travel-notes/ - Travel documentation
    • travelers/ - Traveler profiles and galleries
    • shared/ - Shared utilities, hooks, and components
  • supabase/ - Database migrations and schema
  • public/ - Static assets

How to Get Started

Installation:

  1. Clone the repository or download the project
  2. Navigate to the project directory: cd test-spa
  3. Install dependencies: npm install
  4. Start the development server: npm run dev
  5. Open your browser to http://localhost:5174

Build for Production:

  1. Run: npm run build
  2. Preview: npm run preview
  3. Deploy to your hosting service

Key Features Explained

🎯 Camera Capture Module

Users can capture photos directly from their device camera or upload existing images. The enhanced camera capture provides real-time preview and instant sharing capabilities.

📍 Travel Mapping

Interactive maps show travel destinations, routes, and points of interest. Users can explore geographically and discover new travel inspiration.

🖼️ Gallery System

Beautiful, responsive photo galleries showcase travel moments. Each gallery is optimized for mobile and desktop viewing with smooth animations and zoom functionality.

📝 Travel Notes Publisher

Create detailed travel narratives with text, photos, and metadata. Travel notes are stored securely and can be shared with the community.

⭐ Spotboard

A community-driven platform for sharing hidden gems, travel tips, and recommendations. Users can rate, comment on, and appreciate spots.

Database Schema

The project uses PostgreSQL through Supabase with the following tables:

  • user_profiles - User information and preferences
  • travels - Travel records and metadata
  • photos - Photo storage and references
  • spotboard_photos - Spotboard contributions
  • spotboard_ratings - Community ratings and feedback
  • appreciations - User appreciation/like system

Performance & Security

  • ✅ Row-Level Security (RLS) enabled on all tables
  • ✅ Real-time database updates using Supabase subscriptions
  • ✅ Optimized image loading with responsive sizing
  • ✅ Secure authentication flow with JWT tokens
  • ✅ CORS protection and API rate limiting
